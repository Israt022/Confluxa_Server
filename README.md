# Confluxa Server

Backend API server for **Confluxa - Event Management Platform**.

This server handles authentication, authorization, event management, booking system, database operations, and secure REST APIs using Node.js, Express.js, TypeScript, and MongoDB.

---

## 🚀 Features

- RESTful API architecture
- TypeScript based backend
- MongoDB database integration
- JWT authentication
- Protected API routes
- Role-based authorization
- Event CRUD operations
- Booking management
- User management
- Secure middleware system
- Error handling
